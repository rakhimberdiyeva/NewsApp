**Описание** 

News App — это веб-приложение для публикации и обсуждения новостей.
Проект реализует ролевую модель пользователей и предоставляет удобную платформу
для обмена информацией и мнениями.

**Доступный функционал**
* Регистрация пользователей с использованием библиотеки fastapi-users.
* Аутентификация на основе JWT-токенов.
* Неаутентифицированные пользователи имеют доступ только на чтение.
* Создание комментариев и жалоб доступно только аутентифицированным пользователям.
* Создание статей доступно только пользователям с ролью author и admin.
* Администраторы могут управлять пользователями и изменять их роли.
* Получение подробной информации о текущем пользователе.
* Полный CRUD для категорий.
* Получение подробной информации о статье и выполнение CRUD-операций.
* Получение подробной информации о комментариях и выполнение CRUD-операций.
* Получение подробной информации о жалобах и выполнение CRUD-операций.
* Возможность добавления статей в избранное.
* Возможность добавления изображений к статьям.

**Локальный запуск проекта**

* Склонировать репозиторий:

```commandline
git clone <название репозитория>
```

* Cоздать и активировать виртуальное окружение:

```commandline
py -m venv venv
```
```commandline
.\venv\Scripts\activate
```

* Установить зависимости из файла requirements.txt:
```commandline
pip install -r requirements.txt
```

* Для создания миграций выполнить команду:

```commandline
alembic upgrade head 
```

* Запустить проект:

```commandline
uvicorn main:app --reload
```

* Открыть документацию
```
http://localhost:8000/docs
```
